## 今日学习内容

### 一、深度学习基础概念

| 概念 | 一句话理解 |
|------|-----------|
| **神经元** | 一个小型数学计算单元，接收输入、处理、输出 |
| **层 (Layer)** | 一排神经元，像流水线上的工作站 |
| **神经网络** | 多层神经元组成，能够学习复杂模式 |
| **训练** | 看数据→猜→对错检验→调整参数 的循环过程 |

### 二、深度学习核心原理

**类比：教小孩认猫**
1. 给他看1000张猫的图片
2. 他自己发现规律：尖耳朵、胡须、尾巴...
3. 看到新图片，猜是什么
4. 告诉他对错，他调整判断标准
5. 重复越来越多次，越来越准

**核心公式**
```
预测结果 = 模型(输入数据)
损失 = 比较(预测结果, 真实答案)
调整 = 优化器(损失)
```

### 三、深度学习 vs 机器学习

- 机器学习：更大的范畴，用算法让机器学习
- 深度学习：机器学习的一个分支，用神经网络实现

### 四、大语言模型原理

- **训练数据**：整个互联网的文本（万亿字）
- **学习方式**：预测下一个词
- **本质**：猜错就调整，猜对就加强，重复几十亿次

```
"今天天气" → 猜"不错"
"Hello" → 猜"World"
"深度" → 猜"学习"
```

## 关键知识点

### 深度学习为什么叫"深度"？
因为它有很多层"处理单元"，信息一层一层往上传递，越深越"聪明"

### 调参为什么重要？
就像调试乐器：
- 模型 = 乐器
- 参数 = 弦的松紧
- 调参 = 拧动旋钮直到音准对

### 现代深度学习的变化
| 以前 | 现在(LMM时代) |
|------|--------------|
| 需要大量调参 | 大模型已经调好了 |
| 需要自己设计网络 | 用现成架构 |
| 需要很多数据 | 预训练+微调即可 |
