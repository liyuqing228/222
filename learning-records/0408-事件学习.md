  ---
  Step 3: 深度解析

  1️⃣ 核心定义

  刻意练习是由心理学家Anders Ericsson提出的一种系统化提升技能的方法。

  关键定义：
  刻意练习是一种特殊的学习形式，旨在提升当前表现水平，需要在略微超出当前能力
  范围的任务上进行重复、专注、反馈。

  vs 普通练习的区别：

  | 普通练习   | 刻意练习   |
  |--------|--------|
  | 重复做会的  | 反复做不会的 |
  | 机械重复   | 目标明确   |
  | 无反馈    | 即时反馈   |
  | 低效     | 高效     |
  | 停留在舒适区 | 位于学习区  |

  ---
  2️⃣ 三大区域

          恐慌区
           ▼
     ┌───────────┐
     │  学习区    │  ← 刻意练习发生在这里
     │ (Stretch) │
     └───────────┘
     ┌───────────┐
     │  舒适区    │  ← 普通练习停留在这里
     │ (Comfort) │
     └───────────┘

  为什么学习区最有效？

  - 舒适区：太简单 → 无提升
  - 学习区：有点难但不绝望 → 最佳成长
  - 恐慌区：太难 → 无法进步，可能放弃

  关键：找到"有点难但能搞定"的边界

  ---
  3️⃣ 六大特征

  刻意练习必须满足：

  1. 明确目标 - 具体、可衡量、可达成
  2. 专注投入 - 全神贯注，非自动化执行
  3. 即时反馈 - 知道哪里对、哪里错
  4. 正视错误 - 专注于薄弱环节
  5. 心理表征 - 建立正确的"心智模型"
  6. 逐步改进 - 每次进步一点点

  ---
  4️⃣ 操作方法（四步循环）

      ┌─────────┐
      │ 1. 设定  │
      │   目标  │
      └────┬────┘
           ▼
      ┌─────────┐
      │ 2. 专注  │
      │   练习  │
      └────┬────┘
           ▼
      ┌─────────┐
      │ 3. 获取  │
      │   反馈  │
      └────┬────┘

      ┌─────────┐
      │ 4. 修正  │
      │   改进  │
      └────┬────┘
           │
           └── 循环回到 Step 1

  实践建议：
  - 每天留出1-2小时"刻意练习时间"
  - 一次只聚焦一个技能点
  - 用番茄工作法保持专注

  5️⃣ 常见误区

  | 误区           | 真相                    |
  |--------------|-----------------------|
  | "练够1万小时就行"   | 质量 > 数量，无效练习1万小时还是新手  |
  | "一直做难的事"     | 太难=恐慌区，需要在"稍微超出能力"的范围 |
